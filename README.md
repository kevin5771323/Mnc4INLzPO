# 项目简介

## 核心功能
- 用户管理模块：支持三类用户角色（平台管理员、通报操作员、超级管理员）的权限管理
- 信息爬取模块：自动爬取风险信息数据
- 信息处理模块：对爬取的风险信息进行初步筛选和审核
- 信息推送模块：根据资产情况推送风险通报内容
- 数据管理模块：管理平台所有相关数据

## 适用场景
- 适用于需要进行安全漏洞检测和风险管理的web系统
- 可用于资产安全监控和风险信息通报
- 适合需要自动化处理安全数据的环境

## 技术特点
- 基于Python开发的web安全漏洞渗透测试系统
- 采用模块化设计，包括用户管理、信息爬取、信息处理、信息推送和数据管理模块
- 支持多层次权限管理，确保通报内容准确有效
